Saved
  • opening tab — groups every tab opened from the same parent tab
  • domain — only groups tabs when their domains match the opener's
Using dev.facebook.com as an example:
  • domain — facebook
  • subdomain.domain — dev.facebook
  • subdomain.domain.tld — dev.facebook.com
  • opener's title — the page title of the tab that opened the group
  • nameless — no name
  • 0 — unlimited (default)
  • 2–999 — new tabs won't be added once a group reaches this size
Closing tabs in a group until only one remains will automatically ungroup it.
Switching tabs collapses every group except the one you're in.

Custom Rules

Override group name and color for specific domains.

Blacklist

Tabs opened from blacklisted domains are never grouped.
